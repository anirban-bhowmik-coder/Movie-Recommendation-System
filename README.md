# 🎬 Movie Recommendation System (Logic-Based)

A simple yet effective **logic-based movie recommendation system** built using **Python, Pandas, and Streamlit**. This project demonstrates how recommendations can be generated **without machine learning**, using rule-based scoring techniques.

---

## 🚀 Features

* 📊 Uses a structured dataset of movies
* 🧠 Rule-based recommendation system (no ML required)
* 🎯 Scores movies based on similarity:

  * Same Director (+5)
  * Primary Genre match (+3)
  * Secondary Genre match (+2)
  * Similar Vibe (+1)
* 🌐 Interactive UI built with Streamlit
* ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

* Python
* Pandas
* Streamlit

---

## ⚙️ How It Works

1. User selects a movie from the dropdown
2. The system compares it with other movies
3. A score is calculated based on matching attributes
4. Movies are ranked by score
5. Top 3 recommendations are displayed


---

## 🖥️ UI Usage

* Select a movie you like
* Click **"Find Recommendations"**
* View top 3 recommended movies with match scores
